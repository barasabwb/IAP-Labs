The pictured show how the various routes will be used to access data.You can use http://localhost:4000  to access the api routes.

The defined routes are as follows:
     Create a new student POST request
    http://localhost:4000/api/v1/student

     Retrieve all students GET request
    http://localhost:4000/api/v1/student

     Retrieve a single student by id GET request
    http://localhost:4000/api/v1/student/id

     Update a student with id PUT request
   http://localhost:4000/api/v1/student/id

    Delete a student by id DELETE request
 http://localhost:4000/api/v1/student/id

To insert data in postman use raw JSON format as shown in the pictures.
